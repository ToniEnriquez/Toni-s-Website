Foundations in AI and Data

During my time at Temasek Polytechnic, I built a strong foundation in data analysis, machine learning, and storytelling with data. I worked on a variety of hands-on projects — from using Tableau to extract insights and make business recommendations, to building a laptop price prediction model using a Random Forest Regressor. I also explored text and social media analytics using SAS, clustering customer reviews to uncover sentiment trends in e-commerce.

AI & Image Classification

One of my most impactful projects was an image classification system built using TensorFlow. I developed multiple convolutional neural networks (CNNs) with different activation functions, and later applied transfer learning with pre-trained models like AlexNet, VGG16, and DenseNet using the CIFAR-10 dataset. After training and evaluating the models, VGG16 from ImageNet achieved the highest test accuracy and was selected for Android deployment. This project gave me valuable experience in data preprocessing, model tuning, and real-world implementation of deep learning systems.

Data Storytelling & Healthcare Analytics

In a data storytelling project focused on Singapore’s healthcare system, I used Tableau to uncover key insights about manpower shortages, resource gaps, and demand trends from 2014 to 2019. I created dashboards that visualized nurse-to-doctor ratios, bed availability, and outpatient admissions. The project emphasized the importance of resource allocation in public healthcare and helped me develop a narrative-driven approach to data analysis.

Text Analytics with SAS

Using SAS, I analyzed customer reviews from a women’s fashion e-commerce dataset to uncover sentiment patterns and product feedback. I applied text parsing, concept linking, topic modeling, and clustering techniques to identify common themes like sizing, fabric quality, and visual appeal. This project introduced me to Natural Language Processing (NLP) and showed how text data can reveal valuable consumer insights when explored thoughtfully.

Business Intelligence with MySQL & Tableau

In a business intelligence project, I worked with multiple datasets imported through MySQL and integrated them into Tableau for analysis. I performed data cleaning, calculated fields like age groups and seasonal sales patterns, and used SQL joins to unify tables across customers, products, sales, and stores. The final visualizations explored which age groups purchased specific products during different seasons. This project deepened my understanding of data warehousing, dimensional modeling, and how SQL-based pipelines support BI tools like Tableau in driving data-driven decisions.
